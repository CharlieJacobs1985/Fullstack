HTML

Document Summary

Tag Content Attribute(optional)
Html Your entire HTML <html lang="english"> </html>
The HTML tag will be in every html document. Adding the lang attribute makes it possible for a search robot, to know wich language the website is in.
Head title, reference to stylesheet, ... None
In the head u place things that offer utility to your website but aren't directly visible. A few examples are link to a stylesheet or the title of the page.
Title Document title None
Place the title you want to see on your tab here
Body All visible content Only styling attributes(use CSS)
Everything that is visible on your website is in between the body tags

Document Structure

Semantic

Tag Content Attribute(optional)
Header Introductory content None
Here you place a banner, logo or navigation for example
Nav navigational links None
Often placed in the header or footer, used for navigation on website
Article Text None
The tag says it all, used for writing chunks of text(blog entry, forum post, ...)
Aside indirectly related content None
Quotes, little fun facts, ...
Section Used to divide content in sections None
Devide your webpage in sections with different subjects(website golf devided in apparel and technique)
Main Dominant content None
Here you place the main reason you made your webpage
H1 - H6 Titles or subtitles None
The title of a certain paragraph, article, subject, ...
ol and ul contains the li item for a list None
ordened list and unordened list in the li tag
li List item None
contains list item or a link to document as a list item
Footer Text or links None
Often used for contacts or links to special pages

Non-semantic

Tag Content Attribute(optional)
Div Used to split code into parts None
Use div when no semantic code can be used to divide page
Span Text that needs to be inline None
Used to hihglight a part of a longer text
Br No content None
Used to go to the next line in your content

Links and Images

Tag Content Attribute(optional)
a Used to anchor links None
The anchor tag can contain all kinds of links(hyperlinks, mailadress, phonenumber or id's)
href="hyperlink", href="mailto: emailadress", href="tel: Phonenumber", href="#id". a is also a selfclosing tag
img used to place an image on a page alt="text"
img src="url to img" alt="Text explaining image for when image doesn't load"

Tables

Tag Content Attribute(Optional)
table Announces your gonna make a table on the page None
Start of all table markup
thead The first row of a table that explains the colums in the table None
First row of table
th u place the content of 1 cell in here(in the head) colspan, rowspan
if u use more th's in a row the first one is the first cell in the row and so on
th colspan="#number" The number says how many cells your gonna join together on that row, rowspan does the same for rows
tbody The body starts after the head and contains all tabledata None
All table data is in here
td u place the content of 1 cell in here(in the body) colspan, rowspan
if u use more td's in a row the first one is the first cell in the row and so on
td colspan="#number" The number says how many cells your gonna join together on that row, rowspan does the same for rows
tr adds a new row None
place this tag if u want to go to a new row in a table
